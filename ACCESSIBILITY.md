# Open OnDemand's Accessibility

Accessibility information for Open OnDemand is 
maintained on the Open OnDemand website.

The current Accessibility Conformance Report (ACR), 
accessibility updates, and related documentation are available 
at https://www.openondemand.org/accessibility

Please refer to the website 
for the most up to date details.

If you have accessibility questions or feedback, please 
open an issue in this repository.
